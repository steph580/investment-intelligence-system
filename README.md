# Investment Intelligence System

An automated stock analysis and portfolio optimization system built in Python.

## What It Does
- Analyzes any listed stock using 5 years of financial ratio data
- DCF valuation — identifies if a stock is undervalued or overvalued
- Crash analysis — shows how a company behaved during major market downturns and how fast it recovered
- Portfolio optimization — suggests best position sizing for maximum returns at minimum risk (in progress)

## Built With
- Python
- pandas
- yfinance

## Current Progress
- [x] Financial ratio analysis — profit margin, current ratio, ROE, debt to equity, FCF
- [x] Crash analysis and recovery time modeling
- [ ] DCF valuation automation (in progress)
- [ ] Portfolio optimizer using Markowitz Modern Portfolio Theory
- [ ] Sentiment analysis layer
- [ ] Web dashboard
